# Conch of Safe Rest

*Item, Unknown (Magic)*

You can use an action to blow this pearlescent conch like a [[horn|PHB]], creating an effect identical with that of a [[Leomund's tiny hut]] spell. The conch disappears once the spell takes effect. The effect ends if you leave the spell's area.
