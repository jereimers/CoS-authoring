# Orb of Direction

*Wondrous Item, Common*

This orb can be used as an Arcane Focus.

While holding this orb, you can take a [[Magic|XPHB]] action to determine which way is magnetic north. Nothing happens if the orb is used in a location that has no magnetic north.
